Proceso de Discovery - Sistema de Gestión de Citas en Línea

Contexto del Proyecto
Sistema: Sistema de Gestión de Citas en Línea
Objetivo: Desarrollar una plataforma web que permita a los usuarios agendar, consultar y cancelar citas, y al personal administrativo gestionar la disponibilidad y atención

Stakeholders Identificados
1. Usuario - Personas que necesitan agendar citas
2. Administrador - Personal que gestiona el sistema y usuarios
3. Personal de Atención - Profesionales que brindan el servicio
4. Recepcionista - Personal que apoya en la gestión de citas

Necesidades Identificadas

Gestión de Citas
- Los usuarios desean agendar citas en línea
- Los usuarios quieren ver disponibilidad en tiempo real
- El sistema debe confirmar citas automáticamente
- Se requiere cancelación y reprogramación de citas
- Control de horarios disponibles por profesional

Gestión de Usuarios y Personal
- El administrador debe gestionar usuarios del sistema
- El personal debe gestionar sus horarios de disponibilidad
- Autenticación y perfiles diferenciados por tipo de usuario
- Sistema de notificaciones para cambios en citas

Funcionalidades del Sistema
- El sistema debe notificar cambios de estado de citas
- El sistema debe registrar historial completo de citas
- El personal debe marcar citas como atendidas o no atendidas
- El sistema debe evitar solapamientos de horarios
- Accesibilidad desde navegador web en cualquier dispositivo

Reportes y Administración
- El administrador requiere reportes de uso del sistema
- Estadísticas de citas por período de tiempo
- Control de disponibilidad y utilización de agenda
- Gestión de cancelaciones y no-shows

Restricciones y Limitaciones
- Disponibilidad 24/7 para agendamiento de citas
- Integración con sistema de notificaciones (email/SMS)
- Cumplimiento de normativas de protección de datos médicos
- Soporte para múltiples profesionales y especialidades
- Interfaz intuitiva para usuarios de diferentes edades

Criterios de Éxito
- Reducción del 60% en llamadas telefónicas para agendar citas
- Automatización del 85% de procesos de agendamiento
- Satisfacción del usuario mayor a 90%
- Tiempo de agendamiento menor a 3 minutos
